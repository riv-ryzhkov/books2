Даний проект є навчальним для підтримки курсу з "Django" в Університеті ім.Альфреда Нобеля.


* Важливо! Оскільки сайт створено для навчання та демонстрації, режим DEBUG = True залишено свідомо для контролю роботи сайту.

Основні url, які використовуються в проекті:
''----------> Домашня сторінка
'/admin/'----------> Адмін-сторінка
'/silk/'----------> Статистика SILK
'/about/'----------> Ця сторінка
'/tabs/'----------> Головна таблиця сайту
'/create/'----------> Додавання нової книги
'/book/view/<int:id>/'----------> Перегляд інформації по книзі
'/book/edit/<int:id>/'----------> Редагування інформації по книзі
'/book/delete/<int:id>/'----------> Видалення книги
'/book/new/'----------> Додавання випадкової книги
а також для підтримки API:
'/api/v1/'----------> API перелік книг
'/api/v3/book/'----------> API перелік книг
'/api/v3/book/update/cint:pk>/'----------> API редагування книги (аутентифікація по Token)
'/api/v3/bookdelete/<int:pk>/'----------> API видалення книги

'api/v3/auth/'----------> API авторизація
'api/v3/auth/users/'----------> API перелік користувачів

Командою python manage.py generate <num> додається <num> випадкових книжок для наповнення БД.
